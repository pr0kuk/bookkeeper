Запуск графического интерфейса:
poetry.exe run python bookkeeper/main.py

В верхнем тулбаре есть кнопка Help для подробного описания графического интерфейса
На главном экране два виджета: траты и бюджет
Чтобы открыть виджет категорий, необходимо вызвать контекстное меню, кликнув правой кнопкой мыши по виджету расходов и выбрать добавить
Виджет категорий содержит дерево категорий. Категории создаются и удаляются через контекстное меню.
Чтобы создать расход выбранной категории нажмите кнопку Выбрать
У уже созданных расходов можно менять дату, величну расхода и категорию
В виджете бюджета можно задать дневной/недельный/месячный бюджет и сравнить его со своими тратами
Вся информация сохраняется и загружается при запуске в БД sqlite3 (data/sqlite.db, свой путь можно указать в функции get_rep в bookkeeper/view/presemters.py)
